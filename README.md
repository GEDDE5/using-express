# TinyApp - A URL shortner

## Synopsis

A "full stack" web app built w/ node and express that allows users to shorten long URLS รก la bit.ly

## Installation & Usage


### Install

In terminal:
```js
npm install
npm start
```

### Use

In browser:
```
http://localhost:3000/urls
```

## New features list

* Update the long URL associated w/ the short URL
* Track visitors' IP and access date/time
